# Data formats

* `routes.dat`
    * Airline 
    * Airline ID
    * Source airport
    * Source airport ID 
    * Destination airport 
    * Destination airport ID 
    * Codeshare 
    * Stops 
    * Equipment
* `airports.dat`
    * Airport ID 
    * Name
    * City
    * Country or territory
    * IATA
    * ICAO
    * Latitude
    * Longitude
    * Altitude
    * Timezone
    * DST
    * Tz
    * Type
    * Source
* `us_airports.dat`
    * Flight source
    * Flight destination
    * Number of flights per year
